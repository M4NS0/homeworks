### Passo a passo de instalações: ###
1. Instalar o Node.JS
2. Instalar o Yarn
3. No diretório do projeto:
```sh
$ yarn init
```
4. Instalando dependências do babel
```sh
$ yarn add @babel/cli
$ yarn add babel/preset-env
```
