### SCSS

classes sao utilizados como selectors de estilização
id não é recomendado por ser unico, nao podendo reaproveita-lo
id sao usados para codigo javascript puro

Seletores:
```css
.class {

}

#id {

}

div {

} 
```
```html
<div class="class"></div>
<div id="id"></div>
<div> div selector </div>
```
