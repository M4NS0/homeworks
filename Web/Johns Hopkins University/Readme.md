## Development Enviroment Setup
- Git
- GitHub Account
- Chrome CDT
- Sublime Text 3
- Node JS
- Browser Sync 

### Instalations and Procedures
1. Install node JS Version
2. Check nodeJS Version
```sh
$ npm nodejs --version
``` 
3. Check node Version
```sh
$ npm node --version
``` 
4. Install npm
5. Check npm Version
```sh
$ npm npm --version
``` 
6. Install Browser Sync
```sh
$ npm install -g browser-sync
```
7. Check Browser Sync Version
```sh
$ npm browser-sync --version
``` 
#### Alternative to Browser Sync: 
1. Install live-server
2. Create a test folder with an index.html
```sh
<!DOCTYPE html>
<html lang="en">

    <head>
        <meta charset="UTF-8">
        <title>Document</title>
    </head>

    <body>
        Hello World
    </body>

</html>
```
3. Into test folder, run:
```sh
$ npm install -g live-server
```
3. In same folder, run:
```sh
$ live-server
```
