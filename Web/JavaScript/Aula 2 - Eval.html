Eval

O argumento da propriedade função eval() é uma global string. window Se a string representa uma expressão, eval é uma do objeto eval() avalia a expressão. Se o argumento representa uma ou mais declarações de JavaScript, eval() avalia as declarações. Não chame o eval() para avaliar uma expressão aritmética; JavaScript avalia expressões aritméticas automaticamente. Se você construir uma expressão aritmética como uma string, você pode usar eval() para calcular o resultado depois. Por exemplo, suponha que você tenha uma variável x.
